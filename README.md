# YouTube Clone Backend

Backend API for a YouTube clone built with:

- Node.js
- Express
- TypeScript
- Prisma
- PostgreSQL

## Features

- JWT Authentication
- Video Uploads
- Likes
- Comments & Replies
- Watch History
- Views Tracking
- Playlists
- Subscriptions
- Analytics Dashboard

## Setup

pnpm install

pnpm prisma migrate dev

pnpm dev
